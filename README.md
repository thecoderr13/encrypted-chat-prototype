# Secure Encrypted Chat Prototype

A Python-based encrypted chat application featuring end-to-end encryption using RSA and AES encryption.

## Features

- Secure RSA key exchange
- AES symmetric encryption for messages
- Real-time chat interface
- User list management
- GUI client application

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`

## Usage

### Starting the Server
```bash
cd server
python server.py
